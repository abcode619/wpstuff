== Description ==

A premium WooCommerce plugin that aims to add Social Media Share Buttons at the product page. 
All your visitors now can share content through social media, and it has become the best way to drive traffic.

After you installed the plugin, you need to go to WooCommerce Sidebar Panel, there will be a new menu link called 'Social Buttons PRO' where you can update the options as below:
1. Display Position
2. Enabled Lightbox Button (Lightbox to Show Social Media)
3. Enabled Facebook Share Button
4. Enabled and Sort 10 Options:
	#Facebook
	#Twitter
	#Google+
	#Pinterest
	#LinkedIn
	#StumbleUpon
	#Tumblr
	#Fancy
	#Svpply
	#Email To
5. Select 'Language' for the language for button Facebook(77 languages supported) and Google+(61 languages supported). 

You can also use shortcode [socialbuttonspro] or function socialbuttonspro() to output social buttons in the blog post/page or widget.

NOTE: This plugin requires the WooCommerce Extension and you need to manually enable or sort the social media you prefer after the plugin installation.

== Installation ==

1. Upload the entire *woocommerce-social-buttons-pro* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce panel at left sidebar menu and update the options there.
4. That's it. You're ready to go and cheers!
